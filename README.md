## 🔧 編集方法
### 1. GitHub で直接編集
1. 任意のファイルを開く
2. 右上の ✏️（鉛筆アイコン）で編集
3. 保存すると GitHub Pages に自動反映

### 2. PCで編集 → Gitでpush
ローカル環境で編集して GitHub に push するだけで更新されます。

## 🔍 検索エンジンに表示させたくない場合
```
robots.txt
User-agent: *
Disallow: /
```
これを配置すると検索に出なくなります。
